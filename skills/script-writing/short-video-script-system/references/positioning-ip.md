# Positioning And Personal IP

## Use This File When

- the user says content is scattered
- the account sounds broad, vague, or generic
- the user asks about personal IP, account direction, or content positioning
- script writing is blocked because the speaker identity is still unclear

## Core Principle

Do not start from a giant track.
Start from a clear match:

- who you speak to
- what problem you solve repeatedly
- what value you bring
- what kind of person you are while saying it

Short-video positioning is not a slogan.
It is a repeated promise.

## Personal IP Formula

Personal IP usually needs four things:

1. identity
   Who is speaking, from what stage, with what lived context.
2. point of view
   What do they believe that is worth following.
3. usable value
   What can the viewer take away or do next.
4. stable direction
   What topics will this account keep returning to.

## Positioning Questions

Answer these before writing:

1. Who is the specific audience right now.
2. What real problem are they actively feeling.
3. Why is this speaker qualified to talk about it.
4. What concrete value will the account keep delivering.
5. What action should a new visitor take after 3 seconds on the profile.

## Good Positioning Rules

- one account can be broad in life, but should feel narrow in expression
- one sentence should explain the mainline
- a new visitor should understand the account within the first three profile rows
- keep the same audience, same core problem, and same vocabulary for a period of time

## Positioning Template

Use this line first:

`我在帮 [人群] 用 [方法] 解决 [问题]，重点讲 [持续主题]。`

Examples:

- 我在帮自媒体新手用 AI 辅助提效，解决不会选题、不会写稿、发不稳的问题。
- 我在帮知识型创作者把经验拆成短视频内容，重点讲定位、表达和成交。

## Algorithm Recognition Rules

Platforms identify accounts through repeated signals.
Keep these stable where possible:

- audience language
- root problem
- core keywords
- topic family
- format and tone

If every post changes the audience or promise, recognition gets weaker.

## Positioning Anti-Patterns

- talk about everything because all of it feels related
- use empty labels like growth, self-media, entrepreneurship
- switch audience every few posts
- copy a high-level persona that does not match lived experience
- write a fancy bio that does not match actual content

## Minimum Deliverables

When this stage is the bottleneck, return one or more of:

- one-line positioning statement
- one account mainline
- three content pillars
- seven topic directions under one root problem
- a short profile intro
