# Review Roles

## Review Layers

Each script should be reviewed by these layers in order:

1. structure reviewer
2. Douyin risk reviewer
3. 12 audience-role reviewers
4. scoring reviewer

## Structure Reviewer

Checks:

- one root problem only
- title length and subtitle
- strong first sentence
- real pain scene exists
- one core method only
- one to three steps only
- hard proof anchor exists
- one CTA only

## Douyin Risk Reviewer

Checks:

- no hard follow forcing
- no fake guarantee
- no misleading AI replacement wording
- no direct material bait like “我直接给你 XX”
- no suspicious money promise

## 12 Audience Roles

Run one reviewer per role.

### Role 1: New Creator

Question:

- Will this help a new creator understand the first correction to make?

### Role 2: No Clear Positioning

Question:

- Does this script help someone whose account line is still messy?

### Role 3: Posted But No Follow

Question:

- Does this script explain why someone should keep following this account?

### Role 4: Time-Starved Part-Time Creator

Question:

- Does this feel practical enough for someone with little spare time?

### Role 5: Writes Slowly

Question:

- Does this reduce script friction for someone who can write but is too slow?

### Role 6: Edits Slowly

Question:

- Does this reduce editing friction in a believable way?

### Role 7: AI Curious But Skeptical

Question:

- Does this keep AI in a helper role instead of sounding fake or lazy?

### Role 8: Wants Stable Output

Question:

- Does this make stable output feel more reachable?

### Role 9: Hates Tool搬运 Feel

Question:

- Does this still feel like a real creator speaking, not a tool channel?

### Role 10: Needs Clear Steps

Question:

- Are the actions concrete enough to apply immediately?

### Role 11: Wants Proof

Question:

- Is the proof anchor strong enough to believe the method is grounded?

### Role 12: Looking For Long-Term Value

Question:

- Does this script create a reason to keep watching future content?

## Audience Review Output Format

Each role should answer:

- pass or fail
- strongest sentence
- weakest sentence
- why this role would or would not follow
- exact rewrite advice

## Audience Pass Rule

At least 8 of 12 roles must return positive follow or strong save intent.

If below 8, the script goes back to rewrite.
